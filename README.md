# 🎰 Cassino in Python

Projeto de cassino desenvolvido em Python com sistema de login, saldo persistente e múltiplos jogos.

---

# 🚀 Funcionalidades

* Sistema de cadastro
* Sistema de login
* Saldo salvo no banco de dados
* Jogos disponíveis:

  * Cara ou Coroa
  * Roleta da Sorte
  * Mines
* Sistema de apostas
* Atualização automática do saldo

---

# 🛠️ Tecnologias Utilizadas

* Python
* SQLite3
* Random

---

# ▶️ Como Executar

### 1. Clone o repositório

```bash
git clone https://github.com/Maycondev07/Cassino-in-Python.git
```

### 2. Entre na pasta do projeto

```bash
cd cassino-in-python
```

### 3. Execute o programa

```bash
python main.py
```

---

# 💾 Banco de Dados

O banco de dados é criado automaticamente ao executar o programa.

Arquivo do banco:

```
database.db
```
